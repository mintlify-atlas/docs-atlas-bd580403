RaidBot provides a comprehensive set of commands for managing raid signups, tracking participation, and coordinating events. Commands are organized by category for easy navigation.

## Status & Info Commands

View raid information and bot status.

- [/raidinfo](/commands/raidinfo) - List active raids or view details for a specific raid
- [/stats](/commands/stats) - View participation statistics and analytics
- [/leaderboard](/commands/leaderboard) - View raid leaderboards and achievements
- [/ping](/commands/utility#ping) - Check bot health and response time
- [/changelog](/commands/utility#changelog) - View recent bot updates and new features

## Availability Commands

Record and view when members are available for raids.

- [/availability](/commands/availability) - Set your timezone, preferred days/times, roles, and notes

## Raid Management (Admin)

Create and manage raid signups. Requires Manage Server permission or configured admin role.

- [/create](/commands/create) - Interactive flow to create a raid or museum signup
- [/raid](/commands/raid) - Management panel for raids (close, reopen, delete, change time)
- [/raidsignup](/commands/raidsignup) - Manually assign or remove users from raids

## Recurring Raids (Admin)

Automatic raid scheduling. Requires Manage Server permission or configured admin role.

- [/recurring](/commands/recurring) - Set up recurring raid schedules (weekly, daily, custom)

## Server Setup (Admin)

Configure bot behavior. Requires Manage Server permission or configured admin role.

- [/setchannel](/commands/setchannel) - Set raid, museum, and audit log channels
- [/settings](/commands/settings) - Configure reminders, auto-close, and other options
- [/templates](/commands/templates) - Enable/disable raid templates or rename them
- [/permissions](/commands/permissions) - Set which roles can use admin commands

## Polling (Admin)

Create time slot polls. Requires Manage Server permission or configured admin role.

- [/poll](/commands/poll) - Create reaction-based polls with multiple time options

## Utility Commands

General utility commands available to all users.

- [/help](/commands/utility#help) - Interactive help system with command categories
- [/ping](/commands/utility#ping) - Check bot health and latency
- [/changelog](/commands/utility#changelog) - View recent updates
- [/testalert](/commands/utility#testalert) - Test the alert system (Admin only)
